HTML 基础

超文本标记语言
结构化 Web 网页及其内容的标记语言

元素：开始标签、结束标签与内容的结合，构成一个完整的元素。

元素包含属性。属性：不想在真正的内容中出现，但是又和元素有关的额外信息

嵌套：前端的嵌套一般是 元素的嵌套。一个元素置于其他元素之中

空元素：有开始标签，但是没有内容。例如 img 用属性 src 实现。hr, br 这些都是空元素

HTML文档详解

  • !DOCTYPE html 文档类型，保证文档正确读取
  • html 包含整个页面的所有内容，根元素
  • head 想在html页面中包含，但不想向用户显示的内容的容器。SEO 搜索结果中显示的关键字、页面描述、用于设置页面样式的CSS、字符集声明
  • meta charset="utf-8" 文档使用的字符编码
  • meta name="viewport" content="width=device-width" 视口元素，以宽度进行渲染
  • title 设置页面标题，显示在浏览器标签页上，作为收藏网页的描述文字。alt：可以被屏幕阅读器识别，或者是图片无法显示
  • body 内容

JavaScript 基础

为网站添加交互性的编程语言。

  • 创建游戏
  • 2D、3D图形动画
  • 全面的数据库驱动应用程序
  • WEB 浏览器内置的应用程序编程接口
    • 动态创建HTML
    • 设置CSS样式
    • 从用户的摄像头采集和处理视频流
    • 生成3D图形和音频样本
  • 整合其他万盏的功能到自己的网站中的第三方API
  • HTML加速网站和应用程序开发的第三方框架和库

编程语言的共性

  • 变量：存储值的容器。数据类型
    • 字符串：文本序列
    • 数字
    • 布尔：真/假 true/false
    • 数组：单一引用中存储多个值的结构
    • 对象：一切都是对象
  • 注释： 浏览器会忽略注释
  • 运算符：基于两个值/变量 生成对应结果的数学符号
  • 条件语句：测试表达式的真假
  • 函数：重复使用的功能封装
  • 事件：监听浏览器活动，并在响应中运行代码。
